Put your reference voice bundles here.

Recommended layout:

- `reference_long.wav`
- `reference_long.txt`

The `.txt` file should contain the exact transcript of the reference audio when you use `mode=zero_shot`.

You can also use other supported audio extensions:

- `.mp3`
- `.m4a`
- `.flac`
- `.ogg`
- `.opus`

When several files share the same prefix, the newest audio file wins.
